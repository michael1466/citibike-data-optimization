# CitiBike Data Optimization

## Overview
This project analyzes NYC CitiBike trip data to identify demand patterns and inform potential bike rebalancing strategies.

## Tools
- Python (Pandas, NumPy, Matplotlib)
- Jupyter Notebook

## Key Insights
- Identified peak usage times and high-demand stations
- Observed imbalances in bike distribution across locations
- Highlighted opportunities for rebalancing strategies
- Created simulation van to find optimal routes

## Files
- `citibike_analysis.ipynb`: Full analysis and visualizations
